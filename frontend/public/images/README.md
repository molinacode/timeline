# Imágenes de la aplicación

Coloca aquí los assets según la estructura indicada.  
Guía completa: [docs/guia-imagenes.md](../../docs/guia-imagenes.md).

- **hero/** – Imagen o ilustración del hero de la landing.
- **features/** – Iconos o ilustraciones de las secciones "En un vistazo".
- **timeline/** – Miniaturas de noticias o placeholders.
- **og/** – Imagen para redes sociales (Open Graph), 1200×630 px.

Formato recomendado: WebP (con fallback JPG) para fotos; SVG o PNG para iconos.
